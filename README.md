# JUKEBOX

This project arose as a playground to learn the basics of the Data Sciences with the Python language. It includes several basic statics models such us: Linear Regression, Logistic Regression, etc.

## ARCHITECTURE

1. [Linear Regression](https://github.com/javierpardollama/JUKEBOX/blob/main/ia/linear.ipynb)

2. [Logistic Regression](https://github.com/javierpardollama/JUKEBOX/blob/main/ia/logistic.ipynb)

3. [Multiple Linear Regression](https://github.com/javierpardollama/JUKEBOX/blob/main/ia/multiple_linear.ipynb)

4. [K-Nearest Neighbors](https://github.com/javierpardollama/JUKEBOX/blob/main/ia/nearest_neighbours.ipynb)

5. [Neural Network](https://github.com/javierpardollama/JUKEBOX/blob/main/ia/neural_network.ipynb)

6. [Polynomial Regression](https://github.com/javierpardollama/JUKEBOX/blob/main/ia/polinomial.ipynb)

7. [Decission Tree](https://github.com/javierpardollama/JUKEBOX/blob/main/ia/decission_tree.ipynb)

8. [Random Forest](https://github.com/javierpardollama/JUKEBOX/blob/main/ia/random_forest.ipynb)

9. [Bayesian Regression](https://github.com/javierpardollama/JUKEBOX/blob/main/ia/random_forest.ipynb)
